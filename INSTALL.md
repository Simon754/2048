使用vc6.0以上IDE运行编译文件即可。  
Use IDE above VC6.0 to run compiled files.

无需安装，直接运行exe即可。  
或者利用IDE运行源代码生成exe。  
No need to install, just run exe directly.  
Or use IDE to run source code to generate exe.
